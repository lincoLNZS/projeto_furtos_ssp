# Projeto Furtos SSP 🚔

O repositório é um projeto que partiu de uma curiosidade analítica pessoal. Com o tempo, percebi que pode ser útil também para demais interessados no assunto — especialmente pessoas que deixam suas motocicletas estacionadas na rua por alguns períodos.

## Objetivo
Analisar os dados de furtos de motocicletas no estado de São Paulo, utilizando:
- **Mapa de calor** para mostrar visualmente as regiões com maior número de ocorrências
- Distribuição por **dias da semana**
- Janelas de **horários** com maior incidência

## Estrutura do Projeto
- `analise.ipynb` → Notebook com a análise dos dados  
- `mapa_furtos_interativo.html` → Mapa interativo gerado com Folium  
- `veiculos_subtraidos_2026.xlsx` → Base de dados utilizada  
- `README.md` → Documentação do projeto  

## Tecnologias
- Python  
- Pandas  
- Folium  
- Jupyter Notebook  

## Fonte dos Dados
Secretaria de Segurança Pública de São Paulo (SSP-SP)
